# CRM-Project
<h1>Aquasoft CRM</h1>
<h3>Description:</h3>
<p>
Customer relationship management(CRM) system, designed to be able to see in real-time the status of a lead. The user is able to CREATE, READ, UPDATE and DELETE leads and sales representatives.
